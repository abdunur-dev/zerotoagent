# Zero to Agent Addis

Live Demo: https://02agent-addis.vercel.app/

This is a React-based interactive presentation built for the Zero to Agent Addis Ababa event. It explains the journey from idea to building real AI agents using modern tools like v0, Vercel, and AI SDKs.

Instead of traditional slides, this project is designed as a web experience to make learning more interactive and engaging.

---

## Tech Stack
- React
- JavaScript (ES6+)
- CSS
- Vercel (Deployment)

---

## Features
- Interactive presentation UI
- Component-based React structure
- Smooth and responsive design
- Mobile-friendly layout
- Focused on AI agent education
